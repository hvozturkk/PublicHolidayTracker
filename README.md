# PublicHolidayTracker

Bu proje, C# konsol uygulaması olarak Türkiye'nin 2023, 2024 ve 2025 yıllarındaki resmi tatillerini
https://date.nager.at API'si üzerinden çeker ve kullanıcıya menü ile çeşitli sorgulamalar yapma imkânı sunar.

## Özellikler

- Yıla göre tatil listesini gösterme
- gg-aa formatında tarihe göre tatil arama
- İsme göre tatil arama (yerel isim veya İngilizce isim)
- 3 yılın tüm tatillerini birlikte listeleme (2023–2025)

## Çalıştırma

Bilgisayarınızda .NET 8 SDK kurulu olmalıdır.

```bash
dotnet run
```
